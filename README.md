# Voice Confidence Evaluator

This project analyzes voice recordings from interview responses to estimate a speaker’s confidence level using machine learning techniques.

## Technologies Used

* Python
* Librosa (audio processing)
* Scikit-learn
* NumPy
* Pandas

## Features

* Extracts audio features such as pitch, energy, and speech rate
* Applies machine learning model to estimate confidence level
* Can help candidates analyze interview performance

## Outcome

The system provides a prediction score representing confidence level in spoken responses.

## Future Improvements

* Real-time voice analysis
* Web interface for uploading recordings
